## Установка и запуск приложения

Это приложение — графическая программа для управления расписанием авиаперевозок.  
Ниже описано, как её запустить на обычном компьютере под Windows, не устанавливая отдельно Python и дополнительные библиотеки.

### 1. Что вы получаете

Вы получаете от меня один из вариантов:

- **вариант A**: архив (например, `FlightScheduleGUI.zip`), внутри которого лежит папка с программой;
- **вариант B**: просто файл `FlightScheduleGUI.exe`.

Во всех случаях запуск происходит через `FlightScheduleGUI.exe`.

### 2. Требования к системе

- Операционная система: **Windows 10 / 11, 64‑битная версия**.
- Права: достаточно обычных прав пользователя (администратор не нужен).
- Дополнительное ПО (Python и т.п.) устанавливать **не требуется**.

### 3. Установка

1. Если вы получили архив (`.zip`):
   - сохраните архив в удобную папку (например, `Документы` или `C:\Programs\FlightScheduleGUI`);
   - полностью **распакуйте** архив (не запускать изнутри архиватора);
   - после распаковки у вас должна быть папка, внутри которой лежит файл `FlightScheduleGUI.exe`.
2. Если вы получили уже распакованную папку или один файл `FlightScheduleGUI.exe`:
   - при желании можете просто скопировать её/его в любое удобное место (например, на рабочий стол или в `C:\Programs\FlightScheduleGUI`).

На этом установка завершена: никаких мастеров установки запускать не нужно.

### 4. Первый запуск

1. Откройте папку с программой.
2. Дважды щёлкните по файлу `FlightScheduleGUI.exe`.
3. Возможные сообщения Windows:
   - **SmartScreen / «Windows защитил ваш компьютер»**  
     - нажмите «Подробнее» → «Выполнить в любом случае» (программа подписана мной и не устанавливает ничего в систему);
   - если антивирус спрашивает, разрешить ли запуск, — разрешите.
4. После этого откроется главное окно программы с вкладками:
   - «Расписание», «Рейсы», «Самолёты», «Аэропорты», «Грузы», «Пассажиры», «Отчёты».

Программа не требует подключения к интернету и хранит данные только в памяти во время работы.

### 5. Запуск в дальнейшем

- Чтобы запустить программу в следующий раз, просто снова дважды щёлкните по `FlightScheduleGUI.exe`.
- Для удобства можно:
  - создать ярлык на рабочем столе;
  - закрепить программу на панели задач:
    - запустите её один раз;
    - щёлкните правой кнопкой по значку на панели задач → «Закрепить на панели задач».

### 6. Удаление программы

Программа не прописывается в реестр и не устанавливает драйверов или сервисов.

Чтобы удалить её с компьютера, достаточно:

1. Закрыть приложение (если оно запущено).
2. Удалить папку, в которой лежит `FlightScheduleGUI.exe` (и, при необходимости, архив, из которого вы её распаковывали).
3. При желании удалить созданные вами ярлыки.

После этого программа полностью исчезнет с компьютера.

### 7. Если программа не запускается

Если при запуске появляется окно с текстом об ошибке (про DLL или «приложение не может быть запущено»):

- перезапустите компьютер и попробуйте ещё раз;
- если ошибка остаётся, напишите мне и, по возможности, пришлите:
  - скриншот окна с ошибкой;
  - версию Windows (например, Windows 10 Pro 64‑bit).

В редких случаях на старых системах может потребоваться установка стандартного пакета Microsoft Visual C++ Redistributable — при необходимости я пришлю ссылку и короткую инструкцию.
