# Резюме изменений: Prolog (коммивояжёр) + Python

## Что требовалось

Добавить возможность **вызывать решение задачи коммивояжёра на Prolog из Python**, при этом **условия задачи (расстояния)** должны задаваться **извне**, а не быть зафиксированными фактами в Prolog.

## Что сделано в проекте

### 1) Prolog-решатель без фиксированных `dist/3`

Добавлен файл:

- `python_bindings/prolog/flights.pl`

В нём реализован алгоритм в стиле исходного `flights.pl` (через `edge`, `fly`, `solve_path`, `answer`, `setof/3`), но расстояния берутся **из аргумента `Matrix`**:

- **Точка входа:** `flights:answer(Matrix, BestPath, Dist)`
- **Matrix:** квадратная матрица расстояний \(N \times N\), \(N \le 10\)
- Узлы кодируются числами `0..N-1`

### 2) Python-обёртка для запуска Prolog через `janus_swi`

Добавлен файл:

- `python_bindings/prolog_flights.py`

Содержит:

- `PrologFlightsSolver` — загружает `flights.pl` через `consult/1` и выполняет запрос `flights:answer/3`
- `parse_matrix()` — парсит матрицу из текста (разделители: пробел, запятая, `;`)

Также добавлен авто-поиск `swipl.exe` на Windows и выставление переменной окружения `SWIPL`, если она не задана.

### 3) Вкладка в GUI для запуска решателя

Изменён файл:

- `python_bindings/flight_schedule_gui.py`

Добавлена вкладка:

- **"ЛР5: Коммивояжёр (Prolog)"**

Функциональность вкладки:

- поле ввода матрицы расстояний
- кнопка **"Пример"** (заполняет матрицу 4×4)
- кнопка **"Решить"** (запускает Prolog и выводит лучший путь и длину)

### 4) Документация по запуску

Обновлён файл:

- `docs/дополнительно/QUICK_START.md`

Добавлен раздел про установку SWI‑Prolog + `janus-swi` и использование вкладки в GUI.

## Как запустить

1) Установить SWI‑Prolog (важно: версия, поддерживаемая `janus-swi`).

2) Установить Python‑пакет:

```cmd
pip install janus-swi
```

3) При необходимости задать переменную окружения `SWIPL` (путь к `swipl.exe`), например:

```powershell
setx SWIPL "C:\Program Files\swipl\bin\swipl.exe"
```

4) Запустить GUI:

```cmd
cd Cargo_5th_term\python_bindings
python flight_schedule_gui.py
```

5) Открыть вкладку **"ЛР5: Коммивояжёр (Prolog)"**, вставить матрицу \(N \times N\) и нажать **"Решить"**.
